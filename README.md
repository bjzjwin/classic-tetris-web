# 经典俄罗斯方块游戏

这是一个使用HTML5 Canvas开发的经典俄罗斯方块游戏。游戏完全使用原生JavaScript实现，无需任何外部依赖。

## 游戏特性

- 经典的俄罗斯方块玩法
- 7种不同的方块形状
- 分数系统
- 等级系统（随分数提升自动加速）
- 下一个方块预览
- 流畅的动画效果

## 如何玩

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开始游戏"按钮
3. 使用键盘控制：
   - ↑ 旋转方块
   - ← 向左移动
   - → 向右移动
   - ↓ 加速下落

## 游戏规则

- 使用方向键控制方块的移动和旋转
- 当一行被完全填满时，该行会消失并获得分数
- 每消除一行获得 100 × 当前等级的分数
- 每获得 1000 分升一级
- 等级越高，方块下落速度越快
- 当方块堆到顶部时，游戏结束

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Canvas API

## 本地运行

1. 克隆仓库到本地
2. 在浏览器中打开 `index.html` 文件
3. 开始游戏！
